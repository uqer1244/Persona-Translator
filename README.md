# PersonaASMR-Translator

로컬 Apple Silicon GPU 환경에 최적화된 로컬 VLM (Gemma 4 12B) 기반의 **상황극 및 ASMR 대본 전문 맞춤형 번역 시스템**입니다. 

ASMR 콘텐츠 특유의 섬세한 말투(페르소나)와 작품 배경 상황을 로컬 LLM이 입체적으로 파악하고, 대사 이외의 각종 연출 지시문 및 시간 정보(SRT 타임코드)를 깨뜨리지 않으면서 자연스러운 한국어로 실시간 번역해 줍니다.

---

## 주요 기능

### 1. Apple Silicon GPU 로컬 가속 (MLX VLM)
- Apple Silicon 환경에서 최상의 속도와 효율을 발휘하는 `mlx-vlm` 라이브러리를 탑재했습니다.
- 외부 API 키가 불필요하며, 대용량 대본도 로컬 오프라인 환경에서 유출 우려 없이 안전하고 신속하게 번역합니다.
- `TurboQuant (KV Cache 3.5-bit)` 및 메모리 비우기 자동화 설정을 통해 16GB RAM 사양의 맥에서도 VRAM 부족(OOM) 현상 없이 구동됩니다.

### 2. 다중 파일 및 타임라인 완벽 지원
- **텍스트 입력**: 대본 원문을 웹 화면에 직접 복사-붙여넣기하여 즉시 번역 가능.
- **PDF 텍스트 자동 추출**: `pypdf` 모듈을 내장하여 PDF 포맷의 대본도 바로 올려 번역에 활용할 수 있습니다.
- **SRT 자막 청커**: 자막 번호 및 타임라인(`00:00:01,000 --> 00:00:04,000`)의 포맷 손상 없이 대사만 블록 단위로 문맥을 읽어가며 정교하게 번역합니다.

### 3. 작품 소개 이미지 다중 분석 (VLM 멀티모달)
- 개수 제한 없이 ASMR 작품의 홍보 일러스트, DLsite 트랙 디테일 컷 등을 함께 업로드할 수 있습니다.
- Gemma 4 VLM 모델이 시각 일러스트 이미지와 텍스트를 연계 분석하여 캐릭터의 성격, 외모 분위기, 표정 등을 어투(페르소나) 설정에 정밀하게 반영합니다.

### 4. 대본 설정 및 전체 스토리 요약
- 번역 시작 전 대본과 이미지를 기반으로 다음 정보를 자동으로 추론하여 요약 카드 형태로 시각화해 줍니다.
  - **화자 정보**: 상황극 주인공의 이름 및 호칭
  - **청자 정보**: 듣는 이(청취자)의 역할 및 칭호
  - **배경 상황**: 대본의 핵심 설정 및 장소적 배경
  - **스토리 요약**: 초반 대본을 관통하는 대략적인 전체 시나리오 줄거리

### 5. 페르소나 및 실시간 편집 용어집
- 화자와 청자의 관계 설정, 말투 특징, 5가지 이하의 상세 번역 규칙을 마음대로 조정할 수 있습니다.
- **용어집(Glossary) 실시간 에디터**를 제공하여 고유명사나 특정 ASMR 단어(예: `Darling` -> `자기야`) 번역 고정이 가능합니다. AI 분석 시 적합한 고유명사와 번역어를 용어집 테이블에 자동 기입해 줍니다.

### 6. 실시간 대본 채색 뷰어 (Viewer)
- 실시간으로 스트리밍 번역되는 과정을 읽기 쉽게 종류별로 색깔을 입혀서 출력합니다.
  - `[속삭임]` 등 대괄호 연출 지시문: **굵은 파스텔 오렌지색**
  - `(한숨)` 등 소괄호 감정 지시문: *기울임 파스텔 핑크색*
  - `*소곤소곤*` 효과음 지시문: *기울임 파스텔 민트색*
  - SRT 자막 타임라인: 흐린 monospace 회색

### 7. 말투 및 포맷 자동 교정 (Refiner)
- 번역이 끝난 결과물에 대해 2차 정밀 교정을 진행하여 말투가 뒤섞이는 현상(반말/존댓말 혼용)이나 괄호 쌍 불일치 포맷 오류를 자동으로 완벽하게 다듬어 줍니다.

## 시작 가이드

### 1. 가상환경 및 의존성 설치
터미널을 열고 프로젝트 폴더 내에서 다음 명령어를 차례로 입력하여 Python 가상환경을 구축하고 라이브러리를 설치합니다. (Python 3.11 권장)

```bash
# 가상환경 생성 및 활성화
python -m venv .venv
source .venv/bin/activate

# 의존성 패키지 설치
pip install -r requirements.txt
```

### 2. 로컬 모델 준비
다운로드받은 MLX 포맷 모델 폴더(예: `Gemma4_12B_4bit_mlx` 등)를 프로젝트 하위의 `models/` 디렉토리에 저장합니다.
```
ASMR_ADV/
├── models/
│   └── Gemma4_12B_4bit_mlx/  <-- 다운로드한 모델 폴더
│       ├── config.json
│       ├── model.safetensors (또는 .npz 가중치 파일들)
│       └── tokenizer.json
```
- *참고: 사이드바에서 "직접 경로 입력..."을 선택하면 Mac OS 내 다른 경로에 있는 모델 폴더도 직접 절대경로로 기입하여 로드할 수 있습니다.*

### 3. 애플리케이션 실행
터미널에서 아래의 명령어를 입력하여 Streamlit 로컬 서버를 실행합니다.
```bash
streamlit run app.py
```
- 구동이 완료되면 자동으로 웹 브라우저(`http://localhost:8501`)가 열리며 번역 대시보드를 사용할 수 있습니다.

---

## 기술 스택
- **Core Engine**: `mlx-vlm` (Gemma 4 Unified VLM 아키텍처 연동)
- **Framework**: `Streamlit` (UI 및 백엔드 스레딩 구성)
- **Libraries**: `pypdf`, `pandas`, `transformers`, `huggingface_hub`
